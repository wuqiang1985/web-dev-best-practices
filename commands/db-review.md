# /db-review 命令

调用 **database-reviewer** agent 审查数据库相关代码。

## 审查范围

| 类别 | 检测项 |
|------|--------|
| SQL 质量 | 参数化查询、避免 SELECT *、LIMIT 约束 |
| 索引策略 | 缺失索引、冗余索引、索引列顺序 |
| N+1 查询 | 循环查询、缺少 JOIN/批量查询 |
| 迁移安全 | 可逆迁移、零停机、数据保留 |
| 事务 | 隔离级别、死锁风险、长事务 |
| 连接管理 | 连接池配置、连接泄漏 |
| 数据安全 | 密码加密、权限最小化、审计日志 |

## 使用方式

```
/db-review                         # 审查所有数据库代码
/db-review migrations/             # 审查迁移文件
/db-review src/repositories/       # 审查数据访问层
```

## 输出

数据库审查报告，包含：
- 问题位置和描述
- 修复建议（含代码示例）
- 性能影响评估
- EXPLAIN 分析建议（如适用）
